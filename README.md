# Flask To-Do App

## Overview

A simple and user-friendly To-Do List web application built using Flask and Bootstrap. This application allows users to add and delete tasks through an intuitive interface while demonstrating the fundamentals of Flask web development.

---

## Features

* Add new tasks
* View all tasks
* Delete existing tasks
* Responsive Bootstrap-based user interface
* Flask routing and template rendering
* Lightweight and beginner-friendly project

---

## Technologies Used

* Python
* Flask
* HTML5
* Bootstrap 5

## Installation

### 1. Clone the Repository

git clone YOUR_GITHUB_REPOSITORY_LINK

### 2. Navigate to the Project Directory

cd flask_todo_app

### 3. Install Dependencies

pip install flask

### 4. Run the Application

python app.py

### 5. Open in Browser

http://127.0.0.1:5000

---

## Usage

1. Enter a task in the input field.
2. Click the **Add** button to add the task.
3. View tasks in the list.
4. Click **Delete** to remove a task.
